Overview

The VMStat data source is used to monitor various performance aspects of a *nix host. When the data source starts a process is initiated that runs the vmstat program, and the output from that process is monitored.

Configuration

Every data source requires a Name, which can be any description. The Poll seconds setting determines how often the vmstat process will report data back to Lsscl. The Output scale can be set to automatically adjust byte values to different scales, including bytes, k, KB, m, and MB.
